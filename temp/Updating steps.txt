Package updating steps
1. Add Cpp code to src
  (a) Create header file and point to it
  (b) Create RCPP_MODULE at end of CPP file
2. Use Cpp function in R file via dollarsign
3. Include Roxygen in R file
4. Update namespace with export(R function) if necessary
5. Update description with Rcppmodules name if necessary
7. Comment out the onLoad function in zzz.R
8. Run devtools::document() to update man folder
9. Uncomment out the onLoad function in zzz.R
10. At command line, run R CMD build packagename
11. Within R, run the following to make sure it loads:
    setwd("C:/Dropbox/github")
    detach("package:ngspatial", unload=TRUE)
    install.packages("ngspatial_1.1.tar.gz", repos = NULL, type="source")
    library(ngspatial)
    
    
Note! To build a Windows binary, at the command line run
R CMD INSTALL --build --compile-both ngspatial_1.1.tar.gz

That compile-both flag builds both 32 and 64 bit versions
